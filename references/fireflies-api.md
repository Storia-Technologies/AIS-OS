# Fireflies

Meeting transcript/summary connector. Auth: static API key (no OAuth) from app.fireflies.ai → Settings → Developer Settings. Stored in `.env` as `FIREFLIES_API_KEY` (gitignored).

## Script

`scripts/fireflies_transcripts.py [title_filter] [limit]` — queries the GraphQL API for transcripts, prints date/duration/overview/action items.

## API notes

- Endpoint: `https://api.fireflies.ai/graphql`, `Authorization: Bearer <key>` header
- `transcripts(title, limit)` query — `title` does substring-ish filtering, omit for most recent across all meetings
- Each transcript has `summary.overview` and `summary.action_items` (pre-generated by Fireflies, not raw transcript text)
- For full transcript text/speaker-by-speaker, query `sentences { text speaker_name start_time }` on a specific transcript `id` instead — not implemented in the script yet, add if needed

Full reference: https://docs.fireflies.ai/graphql-api/query/transcript
